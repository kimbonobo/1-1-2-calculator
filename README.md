Getting Started
파이썬 소스코드가 실행 가능한 환경에서 calculator_main.py를 복사 붙여넣기해서 실행한다.

Prerequisites
파이썬 소스코드가 실행 가능한 환경 구축 예) VScode

Give examples
VScode에서 파이썬 프로그래밍 실행가능한 환경을 구축하고 소스코드를 다운받거나 복사 붙여넣기 한 후 실행한다.
더하기 연산을 이용하여 10 + 10 = 20이 나오는지 확인한다.
